ART HANDOFF

巨亨 ONLINE 美術交接文件

這份文件提供給美術、視覺設計與素材協作使用，目標是讓接手的人知道哪些區塊是 CSS 做出來的、哪些地方適合替換圖檔，以及配色與素材建議該怎麼給前端。

視覺方向

目前網站以深紫底、金色 CTA、紫色霓虹感為主，偏娛樂城 / 線上遊戲官網氛圍。

重要觀念

很多看起來像圖的區塊其實是 CSS。改版前先判斷是要換圖檔，還是只要改色票、gradient 與按鈕樣式。

最常需要調整的區塊

  • components/AppHeader.vue：Logo、標題文字、左上角色塊
  • components/BannerSlider.vue：首頁 Banner 背景、標題、說明、按鈕
  • pages/index.vue：快速入口、排行榜快報、熱門活動
  • pages/events.vue：活動卡與活動視覺
  • pages/support.vue、pages/member.vue：資訊卡片與分區樣式

圖檔提供建議

public/images/
public/images/banner/
public/images/logo/
public/images/events/
public/images/icons/
  • Banner 主視覺建議桌機與手機各一版
  • Logo 優先使用 SVG
  • 活動主圖與縮圖建議使用 webp
  • icon 以 SVG 優先
  • 檔名使用英文小寫與 -，避免中文與空白

建議比例

  • Banner 主視覺：16:7 或接近比例
  • 活動卡：16:9
  • 小圖示：正方形

前後端分工建議

  • 美術提供：色票、主視覺圖、活動圖、Logo、icon、卡片背景圖、按鈕狀態規範
  • 前端負責：掛圖、裁切、響應式、路徑處理、互動狀態與整合

目前較偏示意的部分

  • Banner 多數仍是 CSS 視覺
  • 活動卡目前偏資料驅動色塊
  • FAQ / 會員 / 教學頁仍偏資訊卡片風格

建議未來優先補的素材

  • 正式 Logo 套件
  • Banner 桌機 / 手機雙版本
  • 活動主視覺圖與縮圖
  • 排行榜 rank badge 規格
  • CTA 按鈕 hover / active 狀態規範
快速原則：想換整站風格先看 assets/css/main.css；想換 Banner 或活動視覺，請同步看 components/BannerSlider.vue 與 data/siteContent.ts。
